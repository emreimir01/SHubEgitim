Javascript ile neler yapılabilir?

1. Web (Frontend) Uygulamaları

Kullanılan popüler kütüphaneler / frameworkler:

React.js, Vue.js, Angular, Svelte

Bu kütüphaneleri kullanarak SPA (Single Page Application) türünde web uygulamaları geliştirebilirsiniz.

2. Sunucu Tarafı (Backend) Uygulamaları

Javascript, Node.js platformu sayesinde sunucu tarafında da kullanılabilir.

Örnek Teknolojiler:

Node.js, Express.js, Nest.js, Fastify

3. Mobil Uygulamalar

Javascript, native mobil uygulama geliştirme için de kullanılabilir.

Kullanılan Araçlar:

React Native → iOS ve Android için native mobil uygulamalar geliştirmek için kullanılır.
Ionic veya Cordova → HTML, CSS ve Javascript kullanarak web tabanlı mobil uygulamalar geliştirmek için kullanılır.
NativeScript → Native mobil uygulamalar geliştirmek için kullanılır.

4. Masaüstü Uygulamaları

Javascript, HTML ve Css ile masaüstü uygulamaları geliştirmek için de kullanılabilir.

Kullanılan Araçlar:

Electron.js → HTML, CSS ve Javascript kullanarak çapraz platform masaüstü uygulamaları geliştirmek için kullanılır. VSCode, Slack gibi popüler uygulamalar Electron ile geliştirilmiştir.
Tauri → Daha hafif ve güvenli masaüstü uygulamaları geliştirmek için kullanılır.

5. Oyun Geliştirme

Javascript, özellikle 2D oyun geliştirme için uygundur.

Kullanılan Araçlar:

Phaser.js → 2D oyunlar geliştirmek için kullanılan popüler bir kütüphanedir.
Three.js → 3D oyunlar ve grafikler geliştirmek için kullanılan bir kütüphanedir. Babylon.js → 3D oyunlar ve sanal gerçeklik uygulamaları geliştirmek için kullanılan bir kütüphanedir.

6. Makine Öğrenmesi, IoT ve diğer alanlar

  • Makine Öğrenmesi
  • TensorFlow.js ile model eğitimi ve tahminleri
  • >IoT (Nesnelerin İnterneti)
