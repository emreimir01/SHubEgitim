Bootstrap Giriş

Bootstrap, HTML, CSS ve JavaScript kullanılarak yazılmış, açık kaynaklı ve ücretsiz bir front-end kütüphanesidir. Eski Twitter çalışanları Mark Otto ve Jacob Thornton tarafından oluşturulan bu kütüphane, geliştiricilere duyarlı web siteleri oluşturabilme imkanı sağlar. Bootstrap'in sunduğu tasarım şablonları, tipografi, formlar, butonlar ve navigasyon gibi arayüz bileşenleri oluşturmakta kullanılabilmektedir. Bu sayede geliştiriciler, estetik konularla uğraşmadan uygulama geliştirirken, tasarımcılar için de çekici temalar oluşturma konusunda güçlü bir temel sunmaktadır.

Bootstrapi Prjeye Yükleme

  1. Nuget veya npm paket yöneticisi üzerinden
  2. Bootstrap sitesinden indirip manuel yükleme
  3. Hazır cdn kütüphane sitelerinden alacağımız kodu sayfaya ekleyerek kullanabiliriz.

Bootstrap ekranı 12 eşit parçaya bölerek kullanır. Exceldeki grid-ızgara mantığıyla elemanları konumlandırır. Css flex alt yapısını kullanır.

Bootstrap kırılım noktaları

Bootstrap uygulamaya bağlanan ekran çözünürlüğünü algılayarak kullanıcıya farklı responsive tasarımlar sunar.

Containers - Kapsayıcı

container sınıfı kullanarak içeriği ortalayabileceğimiz kapsayıcı bir element oluşturabiliriz.

container fluid sınıfı kullanarak içeriği tüm ekranı kullanan kapsayıcı bir element oluşturabiliriz.
